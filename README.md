# dgsi-lab-G6-2526
Repositorio que utilizaremos para llevar a cabo el desarrollo del laboratorio de la asignatura desarrollo y gestión de sistemas de la información.

Sprint 2 Review 
  En este sprint se completó el modelado del proceso E2E de alta y acceso autónomo del gimnasio HoodGym. Se ah definido el flujo As-Is y el diseño To-Be junto a los KPIs clave (tiempo de onboarding, tasa de abandono y tasa de errores en registro) y sus baselines estimados. Se identificaron riesgos de datos y operativos junto con la métrica de calidad del dato y se elaboró un blueprint conceptual del SIA, definiendo datos maestros, entidades principales y el encaje de CRM, ERP y SCM con sus integraciones.
